# AI Customer Support System (Agentic AI)

## Overview
An intelligent support system that uses Generative AI and agentic workflows to automate customer queries and intelligently escalate critical issues.

## Features
- Query understanding using LLM
- Classification (billing, technical, general)
- Decision engine (auto-resolve, escalate, route, fallback)
- Sentiment-aware prioritization
- Context-aware conversation (planned)

## Tech Stack
- Backend: Node.js + Express
- Frontend: React (planned)
- AI: LLM API

## Goal
To demonstrate an enterprise-grade support automation system using agentic AI principles.
